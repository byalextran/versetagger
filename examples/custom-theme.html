Theme Customization

VerseTagger allows you to fully customize the appearance of verse modals to match your website's design. This page demonstrates several custom themes that you can switch between using the buttons above.

Try hovering over these scripture references with different themes active: John 3:16, Romans 8:28, Psalm 23, Matthew 5:3-10 ESV, and Jeremiah 29:11.

Each theme customizes colors, spacing, and fonts. The warm theme (currently active) uses earthy tones and a serif font for a classic reading experience. Notice how 1 Corinthians 13:4-8 and Philippians 4:6-7 look with different themes!

Creating Your Own Theme

Creating a custom theme is simple. Here's the warm theme currently in use:

const warmTheme = { colors: { // Modal container modalBackground: '#fef3e2', modalBorder: '#e8d4b0', // Text colors textPrimary: '#3d2817', textSecondary: '#5c4028', // Interactive elements linkColor: '#b8651b', linkHoverColor: '#8f4e15', // Verse numbers verseNumberColor: '#a89580' }, spacing: { modalPadding: '20px', modalBorderRadius: '12px', modalMaxWidth: '450px' }, fonts: { fontFamily: 'Merriweather, Georgia, serif', fontSize: '16px', lineHeight: '1.8' } }; const versetagger = new VerseTagger({ proxyUrl: 'https://your-proxy.workers.dev', theme: warmTheme });

More Examples

The Psalms are beautiful with custom themes. Try Psalm 119:105, Psalm 46:1-3, and Psalm 91:1-2 with different visual styles.

Theme Properties Reference

You can customize these aspects of the modal:

Colors

  • modalBackground - Background color of the modal
  • modalBorder - Border color
  • textPrimary - Main text color
  • textSecondary - Secondary text (reference, version)
  • linkColor - Link color
  • verseNumberColor - Color of verse numbers
  • And more - see the Theming Guide

Spacing

  • modalPadding - Internal padding
  • modalBorderRadius - Corner rounding
  • modalMaxWidth - Maximum width

Fonts

  • fontFamily - Font stack to use
  • fontSize - Base font size
  • lineHeight - Line spacing
